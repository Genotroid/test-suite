To get information on how to configure dhtmlxSuite, explore our documentation: https://docs.dhtmlx.com/suite.
You can import dhtmlxSuite into your projects using yarn or npm package manager. To get the trial version of dhtmlxSuite, run the following commands:

// yarn
yarn config set @dhx:registry https://npm.dhtmlx.com
yarn add @dhx/trial-suite

Or

// npm
npm config set @dhx:registry https://npm.dhtmlx.com
npm i @dhx/trial-suite
